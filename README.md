# JustMouth ERP

這是一個基於 Vue 3 + Vite + Google Apps Script API 的簡易 ERP 系統。

功能：
- 庫存查詢（連接 Google Sheet 工作表 '111'）
- 交易查詢（連接 Google Sheet 工作表 '333'）
- PWA 支援，適合手機瀏覽器使用

部署方式：
1. 初始化 GitHub repo
2. 推送程式碼
3. 使用 Vercel 部署
